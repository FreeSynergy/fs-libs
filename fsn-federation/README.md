# fsn-federation

OIDC, SCIM, ActivityPub, and WebFinger federation support for FreeSynergy services.

## Features

| Feature        | Default | Description              |
|----------------|---------|--------------------------|
| `oidc`         | no      | OIDC client support      |
| `scim`         | no      | SCIM provisioning        |
| `activitypub`  | no      | ActivityPub federation   |
| `webfinger`    | no      | WebFinger discovery      |

## Usage

```rust
use fsn_federation::OidcClient;
// See src/lib.rs for full API details.
```
